# 오픈소스 readme 과제
## 리눅스 명령어
### top
### ps

ps 명령어는 현재 시스템에서 돌고있는 프로세스를 보여주는 가장 기본적인 명령어입니다.\
실행 중인 프로세스를 시각화 할 수 있으며, 기본적으로 터미널에서 스시템 프로세스를 조작합니다.
### jops
### kill

리눅스에서 작업을 할 때 응용프로그램이나 프로세스가 멈추는 것을 볼 수 있습니다.\
이 때의 유일한 해결책은 그 프로세스를 종료하는 것입니다.\
대게 프로세스를 죽일 때 쓰입니다. 

툴 형식은 \
`$ kill [options] <pid> [...]`
